# Wiesbaden WebGIS Demo

A demo WebGIS project focused on urban heat and green accessibility analysis for Wiesbaden.

## Tech Stack

- Python
- FastAPI
- PostGIS
- GeoPandas
- React (planned)

## Project Structure

- `backend/` API and backend services
- `pipeline/` ETL scripts
- `frontend/` WebGIS interface
- `data/` raw and processed datasets

## Current Progress

- OSM data extraction (green areas)
- Data cleaning pipeline
- PostGIS integration
- FastAPI endpoint (/green)
